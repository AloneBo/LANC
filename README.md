## APP funaction
* 传输文件(支持后台)
* 传输信息(支持后台)
* 开启FTP服务器(支持后台)
* 自定义头像
* 消息通知
* 复制聊天信息
* 复制FTP链接
* 删除聊天
* 从Bing得到每日壁纸(支持下载,显示到详情页面)
* 下载文件默认到内部Download目录
